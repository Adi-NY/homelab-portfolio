# Service Desk Simulator — ITSM Practice

## Overview
To build hands-on familiarity with enterprise help desk 
workflows before entering a formal IT role, I completed 
structured practice sessions using Service Desk Simulator,
an IT support simulation platform designed to replicate 
real-world ticket management, SLA adherence, and 
end-user communication scenarios.

## Performance Results
| Metric | Result |
|--------|--------|
| Tickets Resolved | 48/50 |
| Resolution Rate | 96% |
| AI Voice Ticket | Resolved |
| Ticket Types | Hardware, Software, Connectivity, Account Access |

## Skills Practiced

### Ticket Triage and Prioritization
- Assessed incoming tickets by severity and business impact
- Prioritized critical issues affecting multiple users 
  over single-user inconveniences
- Managed concurrent tickets without missing SLA targets

### End User Communication
- Responded to simulated end users professionally 
  and empathetically
- Provided clear status updates throughout the ticket lifecycle
- Resolved an AI voice call ticket, practicing real-time calls in a
  verbal technical support under pressure

### Documentation Standards
- Logged accurate resolution notes for each ticket
- Categorized issues correctly for reporting and 
  knowledge base contribution
- Followed escalation procedures for unresolved tickets

### SLA Management
- Maintained awareness of response and resolution 
  time targets
- Prioritized queue based on SLA breach risk
- Achieved 96% resolution rate across full ticket sessions

## Key Takeaway
This simulation reinforced that effective help desk work 
is 50% technical knowledge and 50% communication and 
prioritization. Resolving an AI voice ticket specifically 
required real-time problem diagnosis without the ability 
to ask clarifying questions via text — a skill directly 
applicable to phone-based IT support roles.
